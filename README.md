# 📄 AI PDF Chatbot (RAG Implementation)

Aplikasi berbasis AI yang memungkinkan pengguna untuk melakukan tanya jawab dengan dokumen PDF menggunakan teknik **Retrieval-Augmented Generation (RAG)**.

### 🚀 Fitur
- **PDF Upload:** Membaca data dari file PDF secara lokal.
- **RAG Engine:** Mencari konteks paling relevan menggunakan ChromaDB & HuggingFace Embeddings.
- **LLM Integration:** Menggunakan Google Gemini 1.5 Flash untuk menjawab pertanyaan secara akurat.
- **Streamlit UI:** Antarmuka chat yang modern dan responsif.

### 🛠️ Tech Stack
- **Language:** Python
- **AI Framework:** LangChain
- **Model:** Google Gemini API
- **Vector DB:** ChromaDB
- **UI:** Streamlit
